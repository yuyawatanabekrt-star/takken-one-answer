# 宅建 一問一答トレーナー

宅地建物取引士試験向けの静的Webアプリです。

- `index.html` をブラウザで開くだけで動作します。
- iPhoneではSafariの「ホーム画面に追加」からアプリのように起動できます。
- 直近10年相当の公式過去問PDFへのリンクを掲載しています。
- アプリ内の問題文は、公式問題の原文転載ではなく、学習用に再構成した○×演習です。
- 回答履歴、正答率、復習マークはブラウザの `localStorage` に保存されます。

## iPhoneで使う方法

1. アプリをHTTPSで公開します。自分だけで使う場合は、Netlify、Vercel、GitHub Pagesなどの静的ホスティングで十分です。
2. iPhoneのSafariで公開URLを開きます。
3. 共有ボタンを押します。
4. 「ホーム画面に追加」を選びます。
5. ホーム画面の「宅建一問一答」アイコンから起動します。

ローカルPCの `http://127.0.0.1:5173/` はそのPC専用のURLです。iPhoneに入れるには、iPhoneからアクセスできるHTTPSのURLで公開してください。

## データについて

RETIO公式サイトでは過去の試験問題と正解番号表が公開されていますが、サイト内容の無断転載・改変は制限されています。そのため、このアプリには原文の丸写しではなく、頻出論点を一問一答用に再構成した問題を入れています。
